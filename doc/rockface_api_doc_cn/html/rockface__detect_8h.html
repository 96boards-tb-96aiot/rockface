人脸跟踪，在连续视频帧时使用，会对当前场景的检测的结果分配跟踪ID，连续帧下对同一目标保持相同的跟踪ID

参数
handle[in] 已初始化的Handle（调用 rockface_init_detector 函数初始化）
in_img[in] 输入图像
max_track_time[in] 最大跟踪时间（避免偶然漏检导致目标丢失）
in_track_objects[in] 人脸检测结果
out_track_objects[out] 已跟踪的人脸列表
